Description
===========

GoogRankCmd is a simple command-line utility that retrieves and displays
the Google's PageRank of a Web page ,without using Google Toolbar or any
other browser plugin.



System Requirements
===================


* Windows operating system: Windows 98/ME/2000/XP/2003.
* Internet connection.



Versions History
================


* Version 1.01: Added support for using GoogRankCmd with proxy server
* Version 1.00: First Release.



Using GoogRankCmd
=================

GoogRankCmd doesn't require any installation process or additional DLLs.
In order to use it, just run the executable file - grankcmd.exe with the
the desired Web page addresses as command-line parameters
For Example:
grankcmd http://www.yahoo.com http://www.mozilla.org/products
grankcmd www.aim.com

If you don't have a direct Internet connection, and you want to use
GoogRankCmd through a Web proxy server, you can do it by running
GoogRankCmd with -proxy parameter. For example:
grankcmd -proxy 192.168.0.100:8080 http://www.cnn.com
grankcmd -proxy 192.168.0.10 http://www.nirsoft.net
